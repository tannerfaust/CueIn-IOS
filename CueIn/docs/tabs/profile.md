# Profile Tab

> Goal-setting & life strategy — where users plan their stages, goals, and long-term direction.

---

## Overview

The Profile tab is the strategic layer of CueIn. While other tabs deal with daily and weekly execution, Profile deals with the **big picture** — what the user is working toward and what phase of life they are in.

---

## Key Concepts

| Concept | Description |
|---|---|
| **Stage** | A named life phase the user is currently in (e.g. "College", "Career Transition", "Marathon Prep"). Displayed in the Today tab header |
| **Goals** | Objectives the user is pursuing. Can be tied to specific categories/subcategories |
| **Strategy** | ❓ High-level planning layer — details TBD |
