# Lab Tab

> The workshop — where formulas, mini-formulas, and weekly schedules are created, browsed, and modified.

---

## Overview

The Lab is the design surface of CueIn. Users come here to build new formulas, rearrange their week, browse templates, and manage their formula library. It does **not** run schedules — that's the Today tab's job.

---

## UI Layout (top → bottom)

| Element | Description |
|---|---|
| **Week Overview** | 7 horizontal blocks (Sun → Sat). Each shows the formula(s) assigned to that day. Users can delete or add formulas per day |
| **＋ Button** (top-right) | Opens a creation menu (see below) |
| **Current Formulas** | Section listing all active formulas and mini-formulas with tap-to-edit |
| **Browse Templates** | Opens a separate page of pre-made formulas organized by category |
| **My Folder** | Shows every formula and week preset the user has — active and inactive |

---

## ＋ Button Menu

| Action | Description |
|---|---|
| **Browse Template Library** | Navigate to curated pre-made formulas |
| **New Formula** | Create a full-day formula from scratch |
| **New Mini-Formula** | Create a sub-schedule (e.g. "Morning Routine", "Gym Session") |
| **New Week Preset** | Create a 7-day template mapping formulas to days |

---

## Sections Detail

### Week Overview

- 7 day-columns, each containing one or more formula cards.
- **Tap a formula card** → opens it in the Formula Editor.
- **Swipe / delete** → removes formula from that day.
- **Add** → pick from library or create new.

### Current Formulas

- Lists formulas actively assigned to the current week.
- Tap → opens in [Formula Editor](../features/formula-editor.md).

### Browse Templates

- Separate page with category filtering.
- Templates are read-only until duplicated into the user's library.

### My Folder

- Complete library of user-created and saved formulas.
- Includes active, inactive, full formulas, mini-formulas, and week presets.
- Search / filter support ❓.
